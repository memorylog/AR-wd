Types of AR

Superimposition-based Augmented Reality (AR)

Superimposition based AR provides an ‘alternate’ view of the object in concern, either by replacing the entire view with an augmented view of the object or by replacing a portion of the object view with an augmented view.

In this AR, object recognition plays a vital role because the application cannot replace the original view with an augmented one if it cannot determine what the object is.
